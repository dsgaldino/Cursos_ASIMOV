# Cursos ASIMOV — Python exercises

Personal study notes and challenge solutions from **Asimov Academy** Python courses.

## Contents

```
Python Básico/
├── 01. Material/     Course material and reference solutions
├── 02. Scripts/      Lesson scripts (Jupyter notebooks)
└── 03. Desafios/     Coding challenges (notebooks and .py files)
```

## Highlights

- Control flow, functions, and data structures
- Practical challenges: Caesar cipher, card deck, state guessing game, character counter
- Mix of `.ipynb` notebooks and standalone `.py` scripts

## Requirements

- Python 3.10+
- Jupyter Lab or VS Code with Jupyter extension

## Note

This repository stores **learning exercises**, not a deployable application. For portfolio projects, see the [profile README](https://github.com/dsgaldino/dsgaldino).
